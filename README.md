**Instant Audio Tools** is a powerful Unity Editor extension that lets you edit, preview, and export audio clips directly inside the Unity Editor. With non-destructive editing, real-time waveform preview, and intuitive controls, it's the fastest way to fine-tune audio in your project.

## âœ¨ Features

- **Trim Audio with Markers:** Visually set in/out points using interactive sliders.
- **Apply Fade In/Out:** Smooth out audio transitions with custom fade durations.
- **Adjust Volume (Gain):** Preview changes live with waveform updates.
- **Basic Audio Processing:** Reverse, normalize, noise reduction, mono conversion.
- **Preview in Editor:** Instant playback with real-time audio edits.
- **Save and Export:** Save edited versions of your clip as `.wav` files, or overwrite existing `.wav` clips.

## ðŸŽ® How to Use

4. Hit **Play** to preview changes.
5. Click **Save Audio As** or **Save and Overwrite** (WAV only) to export your edited clip.
